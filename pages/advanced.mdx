# Getting Started with Securescale

Welcome! Follow these steps to get your Securescale environment up and running in minutes.

## 1. Installation

Install the Securescale CLI via your preferred package manager:

```bash
npm install -g @securescale/cli
```

## 2. Configuration

Initialize your project configuration:

```bash
securescale init
```

## 3. Deployment

Deploy your first secure instance:

```bash
securescale deploy
```

For more advanced configurations, check out the sub-topics in this section.
